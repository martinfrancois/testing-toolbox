# WebdriverIO

## Requirements

- NodeJS v20.11.1

## Documentation

- https://webdriver.io/docs/gettingstarted
- https://webdriver.io/docs/allure-reporter/

## How to run

Run tests:

```bash
npm run test
```

Open report:

```bash
npm run report
```

Example report is available in the `allure-report-reference` folder.
