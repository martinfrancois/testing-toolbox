# Jest

## Requirements

- NodeJS v20.11.1

## Documentation

https://jestjs.io/docs/api#2-testeachtablename-fn-timeout

## How to run

Initial setup:

```bash
npm i
```

Run tests:

```bash
npm run test
```

Open report:

```bash
npm run report
```

Example report is available in the `allure-report-reference` folder.
